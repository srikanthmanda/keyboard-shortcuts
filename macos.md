## The One

`Cmd` `Shift` `/`

This shortcut opens search field in the **Help** menu of the open application.
Search for an action and navigate the suggestions to see appropriate menu item
highlighted along with their keyboard shortcuts.

Note: On YouTube, this shortcut opens a list of its keyboard shortcuts.

## Spotlight

- `Cmd` `Space` => Open Spotlight

## Keyboard

- `Ctrl` `Space` => Change keyboard language
- `Cmd` `Ctrl` `Space` => Open emoji & symbols panel (alterative: `Fn` `E`)

### En (–) Dash and Em (—) Dash

- `Opt` `-` => Type En (–) Dash
- `Shift` `Opt` `-` => Type Em (—) Dash

## Application Management

- `Cmd` `O` => Open selected application
- `Cmd` `Q` => Quit application
- `Cmd` `W` => Close application window (without quitting)
- `Cmd` `Esc` => Cycle through application windows
- `Ctrl` `Tab` => Cycle through tabs (for browsers, Finder, text editors, etc.)

## Window Tiling

- `Ctrl` `Fn` `C` => Move window to the centre of the screen
- `Ctrl` `Fn` `F` => Maximize window
- `Ctrl` `Fn` `→` => Snap window to right half of the screen
- `Ctrl` `Fn` `←` => Snap window to left half of the screen

## Multi-tasking

### Managing Applications

- `Cmd` `Tab` => Cycle through open applications
- `Cmd` `Opt` `Esc` => Opens **Force Quit Applications** window

### Desktop Management

- `Ctrl` `↑` => Show Desktops
- `Ctrl` `↓` => Show application windows
- `Ctrl` `→` => Show next Desktop
- `Ctrl` `←` => Show previous Desktop
- `Ctrl` `2` => Go to 2nd Desktop (replace 2 with the Desktop number you want)

## Finder

### Viewing Files and Folders

- `Space` => Preview file (or folders—see [Quick Look Plugins](#quick-look-plugins))
- `Cmd` `1` => Change folder view to icon view
- `Cmd` `2` => Change folder view to list view
- `Cmd` `3` => Change folder view to column view
- `Cmd` `4` => Change folder view to gallery view
- `Cmd` `Shift` `.` => Toggle showing hidden files and folders

### Navigating Folders

- `Cmd` `↓` => Open selected folder (alternative: `Cmd` `O`)
- `Cmd` `↑` => Go up to the enclosing folder
- `Cmd` `[` => Go back
- `Cmd` `]` => Go forward
- `Ctrl` `Tab` => Cycle through Finder tabs

#### Navigating in List View

- `→` => Expand selected folder to list contents
- `←` => Collapse selected folder

### Navigation Shortcuts

- `Cmd` `Shift` `A` => Go to Applications
- `Cmd` `Shift` `O` => Go to Documents
- `Cmd` `Shift` `D` => Go to Desktop
- `Cmd` `Shift` `H` => Go to Home
- `Cmd` `Shift` `G` => Opens **Go to folder** dialogue
- `Cmd` `Opt` `L` => Go to Downloads

## Content Navigation

- `Cmd` `↑` OR `Fn` `←` => Home or go to beginning of the document/webpage
- `Cmd` `↓` OR `Fn` `→` => Go to end of the document/webpage
- `Cmd` `←` OR `Opt` `↑` => Go to beginning of the line
- `Cmd` `→` OR `Opt` `↓` => Go to end of the line
- `Opt` `←` => Go to beginning of the word
- `Opt` `→` => Go to end of the word

> [!TIP]
> Adding `Shift` key to above shortcuts will **SELECT** the text as the cursor
> moves, which you can then copy or cut.

## Fn Key

### Navigation

- `Fn` `←` => Home
- `Fn` `→` => End
- `Fn` `↑` => Page Up
- `Fn` `↓` => Page Down

### Controls

- `Fn` `A` => Toggle (show/hide) Dock
- `Fn` `C` => Toggle Control Centre
- `Fn` `E` => Toggle emoji & symbols panel (alternative: `Ctrl` `Cmd` `Space`)
- `Fn` `F` => Put the open application in fullscreen mode
- `Fn` `H` => Toggle Desktop
- `Fn` `M` => Go to Menu bar (where you can use arrow keys to navigate)
- `Fn` `Del` => Forward Delete (delete character after the cursor)

## Output

### Display

- `Opt` `Shift` `F1` => Decrease brightness in smaller steps
- `Opt` `Shift` `F2` => Increase brightness in smaller steps
- `Opt` `F1` OR `Opt` `F2` => Open Displays settings

### Sound

- `Opt` `Shift` `F11` => Decrease volume in smaller steps
- `Opt` `Shift` `F12` => Increase volume in smaller steps
- `Opt` `F11` OR `Opt` `F12` => Open Sound settings

## Few Tools I Use

### Quick Look Plugins

Quick Look plugins allow you to preview items in Finder. Here are two I use:

- Preview folders: [Folder Quick Look](https://apps.apple.com/app/id6753110395)
- Preview Markdown files: [QLMarkdown](https://github.com/sbarex/QLMarkdown)
- Preview code files: [Syntax Highlight](https://github.com/sbarex/SourceCodeSyntaxHighlight)

More plugins are listed in this repo: [Quick Look plugins](https://github.com/sindresorhus/quick-look-plugins)

### Amethyst

[Amethyst](https://ianyh.com/amethyst) is a tiling window manager for MacOS that
automatically arranges windows for you, in several layouts you can choose from.

Amethyst keyboard shortcuts:

- `Opt` `Shift` `J` => Cycle through applications
- `Opt` `Shift` `Enter` => Make selected application the main application
- `Opt` `Shift` `Space` => Cycle through layouts
- `Ctrl` `Opt` `Shift` `T` => Disable/enable Tiling

Here is a brief guide: [Amethyst: MacOS Tiling Window Manager](https://www.youtube.com/watch?v=7Z9-Ry4yGNc)
